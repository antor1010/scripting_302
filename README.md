# Scripting 302 (Bash)

A collection of **Bash/Shell scripts** focused on building practical scripting skills for automation, Linux workflows, and beginner security-oriented tasks. This repository is designed for learning-by-doing: small scripts, clear logic, and reusable command-line patterns.

## Repository Language

- **Shell (Bash): 100%**

## What’s Inside

This repository includes scripts and examples that may cover:

### Bash Fundamentals
- Variables, input/output, and arguments
- Conditions (`if`, `case`) and loops (`for`, `while`)
- Functions and reusable helpers
- Working with files and directories

### Scripting for Automation
- Batch operations (rename, move, cleanup)
- Log parsing and text processing
- Scheduling-friendly scripts (cron-ready patterns)

### Beginner Cyber Security / Admin Utilities (Optional)
- File permission checks
- Hashing / basic integrity checks
- Quick system inspection helpers (processes, users, network basics)

## Getting Started

### Clone the repository
```bash
git clone https://github.com/antor1010/scripting_302.git
cd scripting_302
```

### Run a script
If needed, make it executable:
```bash
chmod +x <script_name>.sh
```

Run it:
```bash
./<script_name>.sh
```

Or:
```bash
bash <script_name>.sh
```

## Recommended Conventions

When adding scripts, try to include:

- Shebang:
  ```bash
  #!/usr/bin/env bash

- Usage help:
  - Support `-h` / `--help` where it makes sense
- Clear comments and meaningful exit codes
